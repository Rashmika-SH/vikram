# Cosmic Astrology Temple 🔱✨

A futuristic, immersive astrology website featuring 3D animations, interactive cursor effects, and mystical dark theme visuals.

## Features

- 🌌 Full-screen 3D hero with rotating cosmic sphere
- ✨ Custom cursor with glowing trail effects
- 🎨 Dark cosmic theme (black, midnight blue, purple, gold)
- 🌟 Animated starfield background
- 🖐️ Interactive 3D palm reading visualization
- 💫 3D floating service cards with hover effects
- 📊 Animated counters and statistics
- 🎭 Smooth scroll animations with GSAP/Framer Motion
- 📱 Fully responsive design
- 🚀 SEO optimized with Next.js
- 💬 Floating WhatsApp button
- 🌍 International consultation support

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Three.js & React Three Fiber
- Framer Motion
- Tailwind CSS
- GSAP

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Customization

1. Update contact information in `components/Contact.tsx` and `components/WhatsAppButton.tsx`
2. Replace placeholder email/phone with actual details
3. Add real images for blog posts
4. Configure SEO metadata in `app/layout.tsx`
5. Add Google Analytics or tracking scripts
